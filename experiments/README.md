# Experiments Directory

Esta pasta contém os experimentos conduzidos durante o desenvolvimento.

## Estrutura

```
experiments/
├── EXP-001-baseline/       # Primeiro experimento
├── EXP-002-.../            # Experimentos subsequentes
├── logs/                   # Logs de execução
└── notebooks/              # Jupyter notebooks
```

## Convenção de Nomenclatura

Experimentos seguem o formato: `EXP-XXX-nome-descritivo`

Onde:
- `XXX` = Número sequencial (001, 002, etc.)
- `nome-descritivo` = Descrição curta em kebab-case
